# Pathset

IP education and attorney-matching platform for early-stage founders.

## Stack & Deploy
- Next.js, TypeScript, Tailwind → Vercel
- Landing page deploy still pending as of 2026-05-13

## Revenue Model
Free assessment → $50 IP Clarity Session → Pathset Key ($29.99/mo) → DIY filing (~$20)

## Florida Bar Compliance
- Charge client directly, subcontract attorney, keep fee as document prep
- Required footer on all output: "We are not lawyers. We do not provide legal advice."
